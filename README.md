# Tripleten web_project_homeland

## Nome do Projeto

Homeland - De Pátria para Pátria

## Descrição

Página web responsiva da cafeteria Triple Espresso, desenvolvida com HTML5 e CSS. O projeto contempla a criação de uma página web para visualização em diferentes tamanhos de tela, com versionamento e hospedagem em repositório no GitHub.

## Tecnologias e técnicas utilizadas

- HTML5 semântico
- CSS (Flexbox, posicionamento, pseudo classes)
- Uso de media queries
- Metodologia BEM (estrutura de arquivos BEM Flat)
- Uso de Fontes locais na estrutura do projeto
- GitHub
