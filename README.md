# Tripleten web_project_homeland

## Nome do Projeto

Homeland - De Pátria para Pátria

## Descrição

Página web responsiva que mostra as cidades natais de alguns colaboradores da Triple Ten, desenvolvida com HTML5 e CSS. O projeto contempla a criação de uma página web para visualização em diferentes tamanhos de tela, com versionamento e hospedagem em repositório no GitHub e publicação no GitHub Pages para acesso público online.

## Tecnologias e técnicas utilizadas

- HTML5 semântico
- CSS (Flexbox, posicionamento, pseudo classes)
- Uso de media queries
- Metodologia BEM (estrutura de arquivos BEM Flat)
- Uso de fontes locais na estrutura do projeto
- GitHub
